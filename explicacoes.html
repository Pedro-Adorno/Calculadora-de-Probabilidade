Explicações

Média (Distribuição Uniforme): A média representa o valor esperado de uma variável aleatória contínua com distribuição uniforme. A fórmula é (a + b) / 2, onde a é o limite inferior e b o limite superior do intervalo. Ela indica o ponto central do intervalo considerado.

Variância (Distribuição Uniforme): A variância mede a dispersão dos valores em relação à média. Para a distribuição uniforme, a fórmula é (b - a)² / 12. Um valor maior de variância indica maior espalhamento dos dados dentro do intervalo.

Desvio Padrão: O desvio padrão é a raiz quadrada da variância e representa o quanto, em média, os valores se afastam da média. Ele é utilizado para interpretar a variabilidade dos dados de forma mais intuitiva.

Coeficiente de Variação: O coeficiente de variação (CV) é uma medida relativa da dispersão, calculada como (Desvio Padrão / Média) × 100%. Ele permite comparar a variabilidade entre diferentes conjuntos de dados, mesmo com médias distintas.

Probabilidade (Distribuição Uniforme): A probabilidade de uma variável aleatória contínua assumir valores entre x1 e x2 é dada por (x2 - x1) / (b - a), desde que x1 e x2 estejam dentro do intervalo [a, b]. Essa fórmula reflete o fato de que todos os valores dentro do intervalo têm a mesma chance de ocorrer.

Probabilidade (Distribuição Exponencial): Na distribuição exponencial, usamos as fórmulas P(X ≤ x) = 1 - e^(−λx) para a probabilidade acumulada e P(X > x) = e^(−λx) para a probabilidade de sobrevivência. Essa distribuição é comumente usada para modelar o tempo entre eventos em um processo de Poisson, como o tempo até a falha de um equipamento.
