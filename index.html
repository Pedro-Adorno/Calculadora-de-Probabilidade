Bem-vindo ao StatsXplorer

Seja muito bem-vindo ao StatsXplorer, uma plataforma criada com o intuito de realizar cálculos estatísticos de VAC - Distribuição Uniforme e Exponencial.

Este projeto foi desenvolvido por Jonas Santos e Pedro Adorno.

A proposta do StatsXplorer é oferecer uma calculadora para o estudo de distribuições de probabilidade, começando com a Distribuição Uniforme e a Distribuição Exponencial. Aqui, você pode explorar conceitos teóricos, realizar cálculos interativos e visualizar gráficos gerados automaticamente com base nos seus dados.

O site foi construído com linguagens de programação web, como HTML, CSS e JavaScript.
